Notatki są twoją własnością. Wspierane formaty to między innymi: txt, md, html. Dowiedziałeś się czegoś fajnego? - Zapisz to. Podlinkuj w notatce inne notatki. Tworzą się między nimi połączenia, które w podglądzie graficznym przypominają neurony. Właśnie tworzy się mapa twojej wiedzy do której masz szybki dostęp.

Przejdź do Settings -> Editor -> Vim key bindings, aby korzystać ze skrótów Vim.

Stwórz repozytorium Gita w katalogu, w którym trzymasz notatki. Dodaj alias w pliku ~/.bashrc: alias obsidian='cd /home/boziu/Documents/obsidian && git add . && git commit -m "alias commit" && git push' Teraz będąc w katalogu z repozytorium gita, wpisz w shellu obsidian, aby szybko wypushować je do zdalnego repo.
